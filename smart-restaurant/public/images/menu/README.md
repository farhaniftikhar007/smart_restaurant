# Pakistani Food Images

This folder contains images for the Pakistani restaurant menu items.

## Image Files Needed:

1. `1.jpg` - Biryani (Chicken)
2. `2.jpg` - Nihari
3. `3.jpg` - Karahi (Chicken)
4. `4.jpg` - Haleem
5. `5.jpg` - Gulab Jamun
6. `6.jpg` - Lassi (Sweet)
7. `7.jpg` - Seekh Kebab
8. `8.jpg` - Chana Chaat
9. `9.jpg` - Kheer
10. `10.jpg` - Chicken Tikka
11. `11.jpg` - Aloo Paratha
12. `12.jpg` - Mango Lassi
13. `13.jpg` - Jalebi
14. `14.jpg` - Daal Chawal

## Image Requirements:
- Format: JPG or PNG
- Size: 500x300 pixels (recommended)
- Quality: High resolution for best display
- Content: Clear, appetizing photos of the dishes

## How to Add Images:
1. Save your food images with the exact filenames listed above
2. Place them in this folder: `public/images/menu/`
3. The app will automatically display them in the menu

## Fallback:
If an image is missing, the app will show a beautiful placeholder with the dish name.
